# Interactive Message System

## Description
This project is a simple interactive system developed with JavaScript.  
The program asks the user for their name and age, validates the age input, and displays personalized messages according to the user's age.

## Project Structure
- `index.html` → Main HTML file
- `style.css` → Styles file
- `interactive_system.js` → JavaScript logic file
- `README.md` → Project documentation

## Features
- Uses `prompt()` to request user information
- Validates numeric input for age
- Uses `console.error()` for invalid values
- Displays dynamic messages with `alert()`
- Includes comments for code understanding
- Uses `let` and `const` following JavaScript best practices

## How to Run
1. Download or clone the project files.
2. Open the `index.html` file in your web browser.
3. Enter your name and age in the prompt windows.
4. The program will display a personalized message depending on your age.
