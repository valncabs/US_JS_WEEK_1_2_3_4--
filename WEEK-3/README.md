# Dynamic DOM Management with Browser Persistence

## Description
This project is a simple notes application developed with JavaScript.  
Users can add and remove notes dynamically using the DOM API, while Local Storage keeps the notes saved even after refreshing the page.

## Project Structure
- `manipulacion_dom.html` → Main HTML file
- `style.css` → Styles file
- `manipulacion_dom.js` → JavaScript logic file
- `README.md` → Project documentation

## Features
- Adds notes dynamically with `appendChild()`
- Removes notes with `removeChild()`
- Uses `getElementById()` and `querySelector()`
- Updates content with `textContent`
- Saves and retrieves notes using Local Storage
- Displays actions in the console
- Includes comments for code understanding
- Uses `let` and `const`

## How to Run
1. Download the project files.
2. Open the `manipulacion_dom.html` file in your web browser.
3. Write a note in the input field.
4. Click the "Add" button to create a note.
5. Click the "Delete" button to remove a note.
6. Refresh the page to verify that notes remain saved with Local Storage.
