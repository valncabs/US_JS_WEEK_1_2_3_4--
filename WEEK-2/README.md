# Data Management with Objects, Sets and Maps

## Description
This project demonstrates the use of advanced data structures in JavaScript.  
It includes objects, sets, and maps to organize, store, validate, and manipulate product information efficiently.

## Project Structure
- `index.html` → Main HTML file
- `style.css` → Styles file
- `data_management.js` → Main JavaScript file
- `README.md` → Project documentation

## Features
- Creates and validates product objects
- Uses Set to remove duplicate values
- Uses Map to store product categories
- Implements `for...in`, `for...of`, and `forEach()`
- Uses `Object.keys()`, `Object.values()`, and `Object.entries()`
- Displays validation errors with `console.error()`
- Includes comments for code understanding

## How to Run
1. Download the project files.
2. Open the `index.html` file in your web browser.
3. Open the browser console to see the program results.
4. Review the objects, sets, maps, and validations displayed in the console.
